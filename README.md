# 📊 Data Profiler Project -- Customer Churn Analysis

## 📌 Project Overview

This project focuses on Data Preprocessing, Exploratory Data Analysis
(EDA), and Data Profiling for a Customer Churn dataset.\
The objective is to clean, transform, and analyze data before preparing
it for Machine Learning.

------------------------------------------------------------------------

## 🎯 Objective

-   Understand and preprocess raw customer data\
-   Handle missing values and duplicates\
-   Perform Univariate, Bivariate, and Multivariate Analysis\
-   Generate a Data Profiling Report\
-   Prepare dataset for churn prediction modeling

------------------------------------------------------------------------

## 🧠 Project Structure

### Part A -- Fundamentals

-   What is Data Analysis
-   Data Science Project Lifecycle
-   ML Problem Framing (Customer Churn)
-   Introduction to Tensors

### Part B -- Data Acquisition

-   Load CSV using Pandas
-   Convert and Read JSON
-   SQL File Creation & Fetch Queries
-   API Data Fetching

### Part C -- Data Cleaning

-   Missing value handling (Median & Mode Imputation)
-   Duplicate removal
-   Data type correction
-   Dropping irrelevant columns

### Part D -- Exploratory Data Analysis

-   Univariate Analysis (Age, Income, Purchases)
-   Bivariate Analysis (Income vs Churn, Complaints vs Churn)
-   Multivariate Analysis (Correlation Heatmap, Pairplot)

### Part E -- Data Profiling

-   Generated using ydata-profiling
-   Missing value analysis
-   Correlation analysis
-   Data quality warnings

------------------------------------------------------------------------

## 🚀 How to Run

1.  Install required libraries: pip install pandas numpy matplotlib
    seaborn ydata-profiling

2.  Run Jupyter Notebook cells sequentially.

3.  Open generated profiling HTML file in browser.

------------------------------------------------------------------------

## 📊 Key Insights

-   Complaints and Last Purchase Days strongly influence churn.
-   Income and Age show moderate relationship with churn.
-   Missing values were handled appropriately during preprocessing.

------------------------------------------------------------------------

## 📌 Conclusion

The project successfully demonstrates data acquisition, cleaning,
exploratory analysis, and profiling techniques.\
The dataset is well-prepared for building a machine learning churn
prediction model.
